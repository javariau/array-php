Pemahaman tentang Index dalam Array

Apa itu Index?

Index adalah penanda posisi atau lokasi suatu elemen dalam struktur data array. Index biasanya berupa angka integer yang dimulai dari 0 (dalam kebanyakan bahasa pemrograman).

Bayangkan array seperti deretan kotak penyimpanan yang diberi nomor. Nomor kotak itulah yang disebut index, sedangkan isi kotaknya adalah nilai (value) dari array.

Contoh Array

Dalam PHP, kita bisa membuat array seperti ini:

$nama = array("bintang", "joko", "ani", "susi");

Array tersebut dapat divisualisasikan sebagai berikut:

Index 0
"bintang"
Index 1
"joko"
Index 2
"ani"
Index 3
"susi"

Penting untuk Diingat!

Dalam kebanyakan bahasa pemrograman termasuk PHP, index array selalu dimulai dari 0, bukan 1. Jadi elemen pertama memiliki index 0, elemen kedua index 1, dan seterusnya.

Bagaimana Cara Mengakses Elemen Array?

Kita menggunakan index untuk mengakses elemen array. Contoh:

echo $nama[0]; // Output: bintang
echo $nama[2]; // Output: ani

Jika kita mencoba mengakses index yang tidak ada, akan terjadi error.

echo $nama[5]; // Error: undefined offset

Index pada Array Asosiatif

PHP juga mendukung array asosiatif dimana indexnya bukan angka, tetapi string (disebut key).

$mahasiswa = [
    "nama" => "Bintang",
    "umur" => 20,
    "alamat" => "Jakarta"
];

echo $mahasiswa["nama"]; // Output: Bintang
